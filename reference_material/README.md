# Reference Material Manifest

This document provides an inventory of all reference materials available in the papers and textbooks subdirectories.

## Papers

### Apache Spark & PySpark

- **An Introduction to Apache, PySpark and Dataframe Transformations _ by Victor Roman _ Towards Data Science.pdf**
  - Introduction to PySpark fundamentals and dataframe transformation operations for distributed data processing

- **Apache Spark_ A Unified Engine For Big Data Processing.pdf**
  - Overview of Apache Spark's architecture and its role as a unified engine for large-scale data processing

### Data Engineering & ETL

- **2025-12-infographic-etltoolkit-cards.pdf**
  - Visual infographic reference guide for ETL toolkit concepts and best practices

- **Delta Lake_ High-Performance ACID Table Storage over.pdf**
  - Technical paper on Delta Lake's ACID transaction capabilities and table storage architecture

- **Streaming 101_ The world beyond batch – O'Reilly.pdf**
  - Foundational concepts for stream processing and moving beyond traditional batch processing paradigms

- **Streaming 102_ The world beyond batch – O'Reilly.pdf**
  - Advanced streaming concepts and patterns, continuation of Streaming 101

### Machine Learning Operations

- **data validation for machine learning.pdf**
  - Best practices and techniques for validating data quality in machine learning pipelines

- **hidden-technical-debt-in-machine-learning-systems.pdf**
  - Analysis of technical debt that accumulates in ML systems and strategies for managing it

- **rules_of_ml.pdf**
  - Google's best practices and rules for building effective machine learning systems

- **oreilly-ml-ops.pdf** (Note: This appears to be in papers but may be a full textbook)
  - Comprehensive guide to Machine Learning Operations (MLOps) practices and tools

### Data Products & Analytics

- **Designing great data products – O'Reilly.pdf**
  - Principles and methodologies for designing effective data-driven products

- **Practical advice for analysis of large, complex data sets.pdf**
  - Guidelines and best practices for analyzing large-scale, complex datasets

### Critical Perspectives on Data

- **Opinion _ What Data Can't Do - The New York Times.pdf**
  - Critical examination of the limitations and potential misuses of data-driven decision making

- **What Data Can't Do  _ The New Yorker.pdf**
  - Thoughtful perspective on the boundaries of what data and analytics can achieve

## Textbooks

### Apache Spark

- **LearningSpark2.0.pdf**
  - Comprehensive textbook on Apache Spark 2.0, covering fundamentals to advanced topics in distributed data processing

- **big-book-data-engineering.pdf**
  - Extensive guide to data engineering principles, architectures, and best practices

### Delta Lake

- **oreilly-delta-lake_-up-and-running.pdf**
  - Complete guide to implementing and using Delta Lake for reliable data lakes with ACID transactions

### ETL & Data Integration

- **Understanding_ETL_Updated-edition.pdf**
  - In-depth textbook covering Extract, Transform, Load processes and data integration patterns

### Machine Learning Operations

- **oreilly-ml-ops.pdf**
  - Full textbook on implementing MLOps practices for production machine learning systems

## Usage Notes

- Papers are generally shorter, focused articles or research papers (20-50 pages)
- Textbooks are comprehensive references (200+ pages) suitable for deep learning
- All materials are relevant to data science, data engineering, and machine learning coursework
